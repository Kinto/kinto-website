A generic JSON document store for your apps

Store and share application data and synchronize them between user devices.
Kinto exposes an HTTP API to post and retrieve collections of JSON documents.

  • Learn more

Batteries included

Kinto works out the box, providing most of what any application needs to store and consume data.

  • A full-featured Web Administration Console
  • Fine grained permissions
  • User quotas
  • Per-application isolated data containers
  • A synchronization protocol, featuring conflict detection and reporting
  • Learn more

Pluggable by design

And if you miss something from the core, an extensible architecture and many existing plugins can already provide it.

  • File attachments
  • Email & SMS notifications
  • Push notifications support
  • End to end data encryption
  • Data signing and review workflow
  • Learn more
