What is the difference between kinto.js and kinto-client.js?

kinto-client.js is an isomorphic JavaScript HTTP client for the Kinto API, and kinto.js a library for building Offline-first, synchronized Web applications. The former is a dependency of the latter, meaning kinto.js communicates with a Kinto server leveraging the kinto-client.js JavaScript API. kinto-client.js supports a much broader scope of the Kinto API, while kinto.js simply uses what it needs from it to synchronize collection data.
